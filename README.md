# Syntactic Sugar - eCommerce Back End

##### Technologies used (front end):
- Javascript, jQuery, AJAX
- HTML 5, CSS 3
- Bootstrap
- Handlebars

##### Technologies used (back end):
- Node.js, Express.js
- MongoDB
- PostgreSQL
